Welcome to Video streaming server

All the REST API's Endpoints Here

Users routes
  • user register - {domain}/users/register POST
  • login user - {domain}/users/login POST
  • logout user - {domain}/users/logout POST
  • change password - {domain}/users/change-password POST
  • verify token - {domain}/users/refresh-token POST
  • get current user - {domain}/users/current-user GET
  • get user channel details - {domain}/users/c/:userName GET
  • get watch history - {domain}/users/history GET
  • update Cover Image - {domain}/users/cover-image PATCH
Video Routes
  • Publish a video - {domain}/videos/ POST
  • get video by id - /videos/:videoId GET
  • update a video - {domain}/videos/:videoId PATCH
  • delete a video - {domain}/videos/:videoId DELETE
  • Toggle publish status - {domain}/videos/toggle/publish/:videoId PATCH
  • get all video - {domain}/videos/?userId=userId&page=1&limit=10 GET
Tweets
  • Get user Tweets - {domain}/tweets/user/:userId GET
  • create tweets - {domain}/tweets/ POST
  • update user tweet - {domain}/tweets/:tweetId PATCH
  • delete user tweet - {domain}/tweets/:tweetId DELETE
Comments
  • Add comment to video- {domain}/comments/:videoId POST
  • update comment - {domain}/comments/c/:commentId PATCH
  • get video comment - {domain}/comments/:videoId GET
  • delete video comment - {domain}/comments/c/:commentId DELETE
Playlist
  • create Playlist- {domain}/playlists/ POST
  • get user playlist - {domain}/playlists/user/:userId GET
  • get playlist by id - {domain}/playlists/:playlistId GET
  • update user playlist- {domain}/playlists/:userId PATCH
  • delete user playlist - {domain}/playlists/:playlistId DELETE
  • Add video to playlist- {domain}/playlists/add/:videoId/:playlistId PATCH
  • Remove video to playlist- {domain}/playlists/remove/:videoId/:playlistId PATCH
Subscription
  • toggle subscription - {domain}/subscriptions/u/:channelId POST
  • get user channel subscriber - {domain}/subscriptions/u/:channelId GET
  • get subscribed channel - {domain}/subscriptions/c/:subcriberId GET
Like
  • toggle video like - {domain}/likes/toggle/v/:videoId POST
  • toggle comment liker - {domain}/likes/toggle/c/:commentId POST
  • toggle tweet like - {domain}/likes/toggle/t/:tweetId POST
  • get user liked video - {domain}/likes/videos GET
Dashboard
  • get channel dashboard - {domain}/dashboard/stats GET
  • get channel videos - {domain}/dashboard/videos GET
